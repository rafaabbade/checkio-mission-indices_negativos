Enunciado

Carla, uma engenheira de software, precisa acessar elementos específicos de uma lista a partir do final. Escreva uma função que receba uma lista e retorne os últimos três elementos usando índices negativos.

Entrada: Uma lista de inteiros.

Saída: Uma lista com os últimos três elementos.

Exemplo

{% if interpreter.slug == "js-node" %} 
countConsecutiveSummers(42) == 4
